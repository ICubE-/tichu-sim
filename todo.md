### Features
* 방장만 게임 시작, 룰 변경 가능하게 하기
* UX: 커서가 대부분 중앙에서만 움직이게
* 시간 제한 넣기
* 일정 시간 지나면 방 삭제
* 게임 로그가 남게

### Major
* 중요
  * 카드에 문양 넣기
  * 카드 정렬
  * 상대가 무슨 트릭 냈는지 표시
  * 현재 점수 표시
  * 개 트릭 내면 1초 일시정지
  * 라운드 종료되면 점수 표시 전 1초 일시정지
  * 현재 페이즈의 이전 트릭들도 밑에 깔린 카드들로 표시
* 덜 중요
  * 방 내 플레이어 순서 고정하기
  * 팀 배정 가능하게 하기
  * 게임 종료 점수 규칙
  * 게임 중 채팅
  * 게임 종료 후 back to room 버튼 작동하도록 만들기
  * 게임 시작 버튼 4명일 때만 활성화하기

### Minor
* 게임 시작 버튼 위치 옮기기
* 로그아웃 버튼 더 잘 만들기
* 세팅 버튼 더 잘 만들기
* 티츄 페이지 스크롤 되는 거 고치기
* 상대 카드 크기 줄이기
* 측면 상대 카드가 웬만하면 측면에 붙도록 하기
* 상대 정보는 회전시킬 필요 없음
* 위쪽 상대 정보/카드 위치 스위치
* 카드 교환 ui 변경
* 상대 카드 커서 포인터
